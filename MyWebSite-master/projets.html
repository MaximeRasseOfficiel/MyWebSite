Mes Projets

2018 - Application Java - Constitution d'une base de données de Paris Sportifs

Lors de ma première année de mon cursus de Master SID (Sciences et Ingénierie des Données) dispencé à l'UFR des sciences et techniques à Saint Etienne du Rouvray, j'ai pu réaliser un projet de constitution d'une base de données de Paris Sportif.

L'objectif de ce projet est de combiner les compétences complémentaires des étudiants du M1 SID pour créer une application de pronostics sportifs. Les compétences des étudiants se destinant au M2 GIME seront dans un premier temps exploitées pour extraire depuis un site internet un ensemble important de pronostics réalisés par des experts. Ces données devront être organisées et stockées de façon à pouvoir y accéder simplement.

Pour ce projet qui consiste à réaliser un logiciel de web crawling* qui consiste à récupérer des données sur le web pour ensuite les stockés dans une base de données sur laquelle nous allons pouvoir appliquer des traitements.
Pour se faire, nous avons utilisé une librairie de parsing* nomée JSoup qui permet de faire une analyse syntaxique de page web. En récupérant le contenue que nous souhaitions sur les différentes pages Web, nous avons pu constituer une base de données de tous les utilisateurs du site de Paris Sportif ce qui nous permettait par la suite de réaliser de l'apprentissage de données sur celle-ci.

2017 - Application Arduino, Android & Raspberry Pi - Robot piloté par application Android

Dans le cadre de ma licence 3 dispencée à l'UFR des sciences et techniques à Saint Etienne du Rouvray, j'ai pu réaliser un projet de caméra robotisée embarquée et connéctée

L'objectif du projet était de réaliser une caméra motorisée (grâce à un robot) et connéctée à une application Android en temps réel.

Pour réaliser le robot motorisé, j'ai utilisé une base roulante DFRobot 2WD. Le choix de cette base est qu’elle est déjà complète, disposant de deux moteurs pilotables par Arduino, et donc plutôt simple à programmer. D’une base permettant de contenir, la carte Arduino qui permettra de contrôler les moteurs, la carte Raspberry Pi qui va permettre d’envoyer le flux vidéo, le support de piles, pour alimenter la base et les différents modules. Et la caméra qui sera relié au Raspberry pi.

2016 - Application Arduino - Station météo connectée

Dans le cadre d'un projet de ma licence 2 dispencée à l'UFR des sciences et techniques à Saint Etienne du Rouvray, j'ai pu réaliser un projet de station météo connectée.

L'objectif du projet était de réaliser une station météo à base de composants Arduino et de Shield arduino tels que des shields Xbee, Wifi...

Pour réaliser cette station météo connectée, j'ai pu utiliser différents type de capteurs tels que : un capteur d'humidité et de température, un anémomètre, un pluviomètre, la récupérations des données se faisait via une première carte Arduino qui était dite 'Carte Station' car c'est sur celle-ci les différents capteurs étaient branchés, cette carte se trouvait donc dans un boitier à l'extérieur du bâtimment de l'UFR. Une autre carte Arduino dite 'Carte Base', intérieure, disposait d'un Shield LCD pour avoir un écran LCD 2*16 comme celui présenté ci-dessous.

Sur cette écran nous pouvions visualiser les différentes valeurs recueillies par les capteurs. Les données étaient transmises d'un carte Arduino à une autre par l'intermédiaire d'un Emetteur/Recepteur 433MHz. Enfin les données étaient transmise de la Carte Base vers un site Web par l'intermédiaire d'un shield Wifi.
